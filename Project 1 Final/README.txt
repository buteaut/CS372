Instructions for operation
Step 1: Log into flip
Step 2: Navigate to folder holding program files
Step 3: (enter into console) make
Step 4: (enter into console) python3 chatserve.py <port#> <host handle>
Step 5: Open a second instance of flip
Step 6: Navigate to folder holding program files
Step 7: (enter into console) chatclient <servername i.e. flip3, flip2, etc> <port#>
Step 8: In client program enter a user handle
Step 9: Send messages between programs starting with client
Step 10: To disconnect either client or host type "\quit"
Step 11: Press ctrl+c in server program to exit
